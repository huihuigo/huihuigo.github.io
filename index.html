对JDBC的一次学习

首先，我们要认识一些JDBC连接的工具

  1. JDBC API：提供各种操作访问接口，例如：Connection,Statement,PreparedStatement,ResultSet
  2. JDBC DriverManager:管理不同的数据库驱动

JDBC API

基本套路即与数据库建立连接（Connection）,发送SQL语句(Statement),返回处理结果集(ResultSet)

  • DriverManager:管理JDBC驱动
  • Connection:通过DriverManager.getConnection产生，与数据库建立连接.
  • Statement : 通过Connection.createStatement产生，用于发送SQL语句.
  • PreparedStatement: 创建时和创建Statement不同，把SQL语句提前，即connection.prepareStatement(sql);允许预编译,即 sql = "insert into t_booktype values (?,?,?)"，？是占位符，可用PreparedStatement obj.setInt(String name,int value)置值
  • ResultSet : 返回的结果集，由Statement.executeQuery产生

JDBC访问数据库的具体步骤：

  1. 导入驱动，加载具体的驱动类
  2. 与数据库建立连接
  3. 发送SQL语句,执行
  4. 处理结果集

以上为个人的一些皮毛的学习成果，感谢观看。
